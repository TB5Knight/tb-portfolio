## Personal Portfolio Project (React)

## Project Description
This project is my portfolio website, built with React Native, to showcase my skills, projects, and experience. The site is designed to be modern, responsive, and interactive, providing a professional online presence. It also highlights design projects I completed in my mobile development class.


## Features:
- Responsive design for desktop, tablet, and mobile
- Smooth scrolling and clean navigation
- Interactive UI with animations
- Project showcase section with descriptions and links
- Skills/technologies section
- Contact section with relevant details
- Reusable components (Navbar, Footer, etc.)

## setup instructions
1. Create a React Native Project
2. Install all required dependencies, including React, React Navigation, and supporting libraries.
3. Create components folder with required files (Home, About, Projects, Contact, Navbar, Footer)
4. Importing all required modules and dependencies in each component file.
5. The App.js file imports all screen components (Home, About, Projects, Contact) and defines them inside a Stack.Navigator, with "Home" set as the initial route.
6. Coding each file one by one (Navbar,Footer,Home,About,Projects,Contact).
7. Importing the Navbar and Footer on each screen component (Home, About, Projects, Contact).
8. Styling the design via Stylesheet.
9. Testing the design on local host via npm start.
10. Publishing design on GitHub Pages.
